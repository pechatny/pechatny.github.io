# Как подключить Perplexity в IntelliJ IDEA как AI ассистента

Для интеграции Perplexity AI в IntelliJ IDEA воспользуйтесь следующим пошаговым руководством.

---

## Шаг 1. Регистрация в Perplexity

Если у вас ещё нет аккаунта, зарегистрируйтесь на сайте:

[https://www.perplexity.ai](https://www.perplexity.ai)

---

## Шаг 2. Получение API ключа

1. Войдите в свой аккаунт на Perplexity.
2. Перейдите в раздел **Аккаунт** → секция **API**.
3. Сгенерируйте новый API ключ.

---

## Шаг 3. Установка плагина ProxyAI в IntelliJ IDEA

1. Откройте IntelliJ IDEA.
2. Перейдите в **File** → **Settings** (или **Preferences** на macOS) → **Plugins**.
3. Найдите плагин **ProxyAI** и установите его.
4. Перезапустите IDE для применения изменений.

---

## Шаг 4. Настройка плагина ProxyAI для Perplexity

1. В настройках плагина перейдите в раздел **Providers** → выберите **Custom OpenAI**.
2. Нажмите на кнопку **+** для добавления нового провайдера.
3. В поле **Preset Template** выберите **Perplexity AI**.
4. Вставьте API ключ, полученный на шаге 2, в поле **API key**.
5. Настройте секцию **Chat Completions**:
   - Внизу найдите секцию **Header и Body**.
   - Выберите **Body**.
   - В поле `model` замените значение на `sonar`.
6. Аналогично настройте секцию **Code Completions**:
   - Выберите **Body**.
   - Замените значение `model` на `sonar`.

---

## Шаг 5. Использование

1. Сохраните настройки.
2. В чате плагина выберите созданного провайдера Perplexity.
3. Начинайте пользоваться AI ассистентом для помощи в кодировании.
